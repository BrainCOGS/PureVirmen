# PureVirmen Repository

Functional ViRMEn repository with transformation function for mini Rigs

ViRMEn engine and experiments used in the Accumulating Towers task.


## INSTALLATION STEPS

- Place the PureViRMEn folder anywhere on your computer.
- Add the PureViRMEn folder, with subfolders, to the MATLAB path.
- Run >> virmen at the command line in MATLAB.
- Select Experiment -> Open from ViRMEn GUI.
- Select "live_calibration" from experiment list.
- Click on run button.
- Follow these [instructions](https://braincogs.github.io/maintenance/projection.html#projection-calibration) to complete calibration.
